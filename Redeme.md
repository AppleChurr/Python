    동일한 경로 상의 영상 파일에 대한 .idx 파일 생성

### 0.9.1
    - .avi, .mkv도 사용 가능

### 0.9.0
    - 해당 .exe 파일과 동일한 경로에 .mp4 동영상을 넣고 실행시키면, IDX 파일을 생성
    - 해당 경로상에 여러개의 동영상을 넣어서 실행하면, 여러개의 IDX 파일이 바로 생성
    - 해당 .exe파일을 .mp4 동영상이 있는 폴더로 옮겨서 사용하셔도 가능
